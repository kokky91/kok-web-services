Restaurant instellingen

🖨️ Bluetooth printer (ESC/POS)
Werkt met 58mm/80mm Bluetooth ESC/POS printers. Vereist Chrome/Edge op Android of desktop — iOS Safari ondersteunt Web Bluetooth niet.
🔐 Eigenaar-PIN
4 cijfers. Standaard is 0000 — verander 'm asap. Verplicht bij elke kassa-opening na 8u idle.
🔗 Reserveer-link voor op je website
Zet deze link op je restaurant-website (knop "Reserveren"). Boekingen verschijnen automatisch onder het kopje Reserveringen hierboven.
☁ API server (sync tussen devices)
Draait de lokale server op het restaurant? Start hem met python lan-server.py en bestelt ook zonder internet tussen alle devices op jullie WiFi.
📱 QR basis-URL (naar bestel-pagina)
⚠️ QR werkt niet op telefoons — deze URL wijst naar localhost. Gebruik LAN IP (zelfde WiFi) of je publieke domein.
Dit is de basis van elke QR-code — tafel-QR's krijgen ?r=…&t=N eraan, afhaal krijgt ?m=takeaway.
